## **开始使用**

要在你的 CrewAI 项目中使用任何工具：

1. **导入** crew 配置中的工具
2. **添加**它到你的智能体工具列表
3. **配置**任何所需的 API 密钥或设置

```python
from crewai_tools import FileReadTool, SerperDevTool

# 将工具添加到你的智能体
agent = Agent(
    role="研究分析师",
    tools=[FileReadTool(), SerperDevTool()],
    # ... 其他配置
)
```

## **最大使用次数**

你可以为工具设置最大使用次数，以防止它被使用超过特定次数。
默认情况下，最大使用次数是无限制的。

```python
from crewai_tools import FileReadTool

tool = FileReadTool(max_usage_count=5, ...)
```
