## 概述

测试是开发过程的关键部分，对于确保您的团队按预期执行至关重要。使用 crewAI，您可以轻松测试您的团队并使用内置测试功能评估其性能。

### 使用测试功能

我们添加了 CLI 命令 `crewai test` 使测试您的团队变得容易。此命令将为指定迭代次数运行您的团队，并在运行结束时提供详细的性能指标。参数是 `n_iterations` 和 `model`，它们是可选的，默认分别为 2 和 `gpt-4o-mini`。目前，唯一可用的提供商是 OpenAI。

```bash
crewai test
```

如果您想运行更多迭代或使用不同的模型，可以像这样指定参数：

```bash
crewai test --n_iterations 5 --model gpt-4o
```

或使用简短形式：

```bash
crewai test -n 5 -m gpt-4o
```

当您运行 `crewai test` 命令时，团队将为指定迭代次数执行，并在运行结束时显示性能指标。
