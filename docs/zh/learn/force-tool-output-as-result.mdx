## 介绍

在 CrewAI 中，您可以强制将工具的输出作为代理任务的结果。
此功能很有用，当您希望确保工具输出被捕获并作为任务结果返回时，避免在任务执行期间代理进行任何修改。

## 强制工具输出作为结果

要强制将工具输出作为代理任务的结果，您需要在向代理添加工具时将 `result_as_answer` 参数设置为 `True`。
此参数确保工具输出被捕获并作为任务结果返回，而无需代理进行任何修改。

以下是如何强制将工具输出作为代理任务结果的示例：

```python Code
from crewai.agent import Agent
from my_tool import MyCustomTool

# 创建带有自定义工具的编码代理
coding_agent = Agent(
        role="Data Scientist",
        goal="Produce amazing reports on AI",
        backstory="You work with data and AI",
        tools=[MyCustomTool(result_as_answer=True)],
    )

# 假设工具的执行和结果填充在系统内发生
task_result = coding_agent.execute_task(task)
```

## 工作流程实战

<Steps>
   <Step title="任务执行">
      代理使用提供的工具执行任务。
   </Step>
   <Step title="工具输出">
      工具生成输出，被捕获为任务结果。
   </Step>
   <Step title="代理交互">
      代理可能从工具中反思和学习，但输出不被修改。
   </Step>
   <Step title="结果返回">
      工具输出作为任务结果返回，无需任何修改。
   </Step>
</Steps>
