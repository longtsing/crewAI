CrewAI 支持与 OpenAI 的 DALL-E 集成，允许您的 AI 代理作为其任务的一部分生成图像。本指南将引导您了解如何在 CrewAI 项目中设置和使用 DALL-E 工具。

## 先决条件

- 已安装 crewAI（最新版本）
- 具有访问 DALL-E 权限的 OpenAI API 密钥

## 设置 DALL-E 工具

<Steps>
    <Step title="导入 DALL-E 工具">
        ```python
        from crewai_tools import DallETool
        ```
    </Step>

    <Step title="将 DALL-E 工具添加到您的代理配置">
        ```python
        @agent
        def researcher(self) -> Agent:
            return Agent(
                config=self.agents_config['researcher'],
                tools=[SerperDevTool(), DallETool()],  # 将 DallETool 添加到工具列表
                allow_delegation=False,
                verbose=True
            )
        ```
    </Step>
</Steps>

## 使用 DALL-E 工具

一旦您将 DALL-E 工具添加到您的代理，它就可以基于文本提示生成图像。该工具将返回生成图像的 URL，可以在代理的输出中使用或传递给其他代理进行进一步处理。

### 示例代理配置

```yaml
role: >
    LinkedIn 个人资料高级数据研究员
goal: >
    根据提供的姓名 {name} 和域 {domain} 揭示详细的 LinkedIn 个人资料
    基于域 {domain} 生成 Dall-e 图像
backstory: >
    您是经验丰富的研究员，擅长发现最相关的 LinkedIn 个人资料。
    以高效浏览 LinkedIn 而闻名，您擅长收集和呈现
    专业信息清晰简洁。
```

### 预期输出

带有 DALL-E 工具的代理将能够生成图像并在其响应中提供 URL。然后您可以下载图像。

## 最佳实践

1. **在您的图像生成提示中具体化** 以获得最佳结果。
2. **考虑生成时间** - 图像生成可能需要一些时间，因此请将其纳入您的任务计划中。
3. **遵循使用策略** - 生成图像时始终遵守 OpenAI 的使用策略。

## 故障排除

1. **检查 API 访问权限** - 确保您的 OpenAI API 密钥具有访问 DALL-E 的权限。
2. **版本兼容性** - 检查您是否使用最新版本的 crewAI 和 crewai-tools。
3. **工具配置** - 验证 DALL-E 工具已正确添加到代理的工具列表中。
