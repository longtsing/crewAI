# 在 CrewAI 中设置特定代理作为管理器

CrewAI 允许用户设置特定代理作为团队的管理器，为任务的管理和协调提供更多控制。
这个功能使管理角色的定制化能够更好地适应您项目的需求。

## 使用 `manager_agent` 属性

### 自定义管理代理

`manager_agent` 属性允许您定义一个自定义代理来管理团队。这个代理将监督整个过程，确保任务高效完成并达到最高标准。

### 示例

```python Code
import os
from crewai import Agent, Task, Crew, Process

# 定义您的代理
researcher = Agent(
    role="研究员",
    goal="对 AI 和 AI 代理进行彻底的研究和分析",
    backstory="您是专家研究员，专攻技术、软件工程、AI 和初创公司。您作为自由职业者工作，目前正在为新客户进行研究。",
    allow_delegation=False,
)

writer = Agent(
    role="高级写手",
    goal="创建关于 AI 和 AI 代理的引人入胜的内容",
    backstory="您是高级写手，专攻技术、软件工程、AI 和初创公司。您作为自由职业者工作，目前正在为新客户编写内容。",
    allow_delegation=False,
)

# 定义您的任务
task = Task(
    description="生成 5 个有趣的文章创意列表，然后为每个创意写一个引人入胜的段落，展示此主题下完整文章的潜力。返回带有段落和注释的创意列表。",
    expected_output="5 个要点，每个都有一个段落和附带的注释。",
)

# 定义管理代理
manager = Agent(
    role="项目经理",
    goal="高效管理团队并确保高质量的任务完成",
    backstory="您是经验丰富的项目经理，擅长监督复杂项目并引导团队取得成功。您的角色是协调团队成员的努力，确保每个任务按时完成并达到最高标准。",
    allow_delegation=True,
)

# 使用自定义管理器实例化您的团队
crew = Crew(
    agents=[researcher, writer],
    tasks=[task],
    manager_agent=manager,
    process=Process.hierarchical,
)

# 开始团队的工作
result = crew.kickoff()
```

## 自定义管理代理的好处

- **增强控制**：量身定制管理方法以适应您项目的特定需求。
- **改进协调**：通过经验丰富的代理确保高效的任务协调和管理。
- **可定制管理**：定义与您项目目标一致的管理角色和职责。

## 设置管理器 LLM

如果您正在使用分层流程并且不想设置自定义管理代理，您可以为管理器指定语言模型：

```python Code
from crewai import LLM

manager_llm = LLM(model="gpt-4o")

crew = Crew(
    agents=[researcher, writer],
    tasks=[task],
    process=Process.hierarchical,
    manager_llm=manager_llm
)
```

<Note>
使用分层流程时，必须设置 `manager_agent` 或 `manager_llm`。
</Note>
