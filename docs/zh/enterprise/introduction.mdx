## 介绍

CrewAI AOP（代理操作平台）提供了一个平台，用于在生产环境中部署、监控和扩展您的团队和代理。

<Frame>
  <img src="/images/enterprise/crewai-enterprise-dashboard.png" alt="CrewAI AOP Dashboard" />
</Frame>

CrewAI AOP 扩展了开源框架的强大功能，专为生产部署、协作和可扩展性而设计。将您的团队部署到托管基础设施并实时监控其执行。

## 核心功能

<CardGroup cols={2}>
  <Card title="团队部署" icon="rocket">
    通过几次点击将您的团队部署到托管基础设施
  </Card>
  <Card title="API 访问" icon="code">
    通过 REST API 访问您部署的团队，与现有系统集成
  </Card>
  <Card title="可观察性" icon="chart-line">
    使用详细的执行跟踪和日志监控您的团队
  </Card>
  <Card title="工具存储库" icon="toolbox">
    发布和安装工具以增强您的团队能力
  </Card>
  <Card title="Webhook 流式传输" icon="webhook">
    将实时事件和更新流式传输到您的系统
  </Card>
  <Card title="团队工作室" icon="paintbrush">
    使用无代码/低代码界面创建和自定义团队
  </Card>
</CardGroup>

## 部署选项

<CardGroup cols={3}>
  <Card title="GitHub 集成" icon="github">
