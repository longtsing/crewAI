## 文字教程
<Note>
  **Python 版本要求**

  CrewAI 需要 `Python >=3.10 and <3.14`。以下是检查版本的方法：
  ```bash
  python3 --version
  ```

  如果需要更新 Python，请访问 [python.org/downloads](https://python.org/downloads)
</Note>

<Note>
  **OpenAI SDK 要求**

  CrewAI 0.175.0 需要 `openai >= 1.13.3`。如果您自己管理依赖项，请确保您的环境满足此约束以避免导入/运行时问题。
</Note>

CrewAI 使用 `uv` 作为其依赖管理和包处理工具。它简化了项目设置和执行，提供无缝体验。

如果您还没有安装 `uv`，请按照 **步骤 1** 在您的系统上快速设置，否则可以跳到 **步骤 2**。

<Steps>
    <Step title="安装 uv">
      - **在 macOS/Linux 上：**

        使用 `curl` 下载脚本并用 `sh` 执行：

        ```shell
        curl -LsSf https://astral.sh/uv/install.sh | sh
        ```
        如果您的系统没有 `curl`，可以使用 `wget`：

        ```shell
        wget -qO- https://astral.sh/uv/install.sh | sh
        ```

      - **在 Windows 上：**

        使用 `irm` 下载脚本，使用 `iex` 执行：

        ```shell
        powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
        ```
        如果遇到任何问题，请参考 [UV 安装指南](https://docs.astral.sh/uv/getting-started/installation/) 获取更多信息。
    </Step>

    <Step title="安装 CrewAI 🚀">
      - 运行以下命令安装 `crewai` CLI：
        ```shell
        uv tool install crewai
        ```
          <Warning>
            如果遇到 `PATH` 警告，运行此命令更新您的 shell：
            ```shell
            uv tool update-shell
            ```
          </Warning>

          <Warning>
            如果在 Windows 上遇到 `chroma-hnswlib==0.7.6` 构建错误（`fatal error C1083: Cannot open include file: 'float.h'`），请安装 [Visual Studio Build Tools](https://visualstudio.microsoft.com/downloads/) 并选择 *Desktop development with C++*。
          </Warning>

      - 要验证 `crewai` 是否已安装，运行：
        ```shell
        uv tool list
        ```
      - 您应该看到类似这样的输出：
        ```shell
        crewai v0.102.0
        - crewai
        ```
      - 如果需要更新 `crewai`，运行：
        ```shell
        uv tool install crewai --upgrade
        ```
      <Check>安装成功！您准备好创建第一个团队了！ 🎉</Check>
    </Step>
</Steps>

# 创建 CrewAI 项目

我们推荐使用 `YAML` 模板脚手架来结构化地定义代理和任务。以下是开始的方法：

<Steps>
  <Step title="生成项目脚手架">
    - 运行 `crewai` CLI 命令：
      ```shell
      crewai create crew <your_project_name>
      ```

    - 这将创建一个具有以下结构的新项目：
      ```
      my_project/
      ├── .gitignore
      ├── knowledge/
      ├── pyproject.toml
      ├── README.md
      ├── .env
      └── src/
          └── my_project/
              ├── __init__.py
              ├── main.py
              ├── crew.py
              ├── tools/
              │   ├── custom_tool.py
              │   └── __init__.py
              └── config/
                  ├── agents.yaml
                  └── tasks.yaml
      ```
  </Step>

  <Step title="自定义您的项目">
    - 您的项目将包含这些基本文件：
      | 文件 | 用途 |
      | --- | --- |
      | `agents.yaml` | 定义您的 AI 代理及其角色 |
      | `tasks.yaml` | 设置代理任务和工作流 |
      | `.env` | 存储 API 密钥和环境变量 |
      | `main.py` | 项目入口点和执行流 |
      | `crew.py` | 团队编排和协调 |
      | `tools/` | 自定义代理工具目录 |
      | `knowledge/` | 知识库目录 |

    - 首先编辑 `agents.yaml` 和 `tasks.yaml` 来定义您的团队行为。
    - 将 API 密钥等敏感信息保存在 `.env` 中。
  </Step>

  <Step title="运行您的团队">
    - 在运行团队之前，确保运行：
      ```bash
      crewai install
      ```
    - 如果需要安装额外的包，使用：
      ```shell
      uv add <package-name>
      ```
    - 要运行您的团队，在项目根目录执行以下命令：
      ```bash
      crewai run
      ```
  </Step>
</Steps>
